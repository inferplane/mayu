team governance

Rate/TPM/quota/budget limits are enforced live on every request for this team the moment you save — no restart.

Counters are enforced per gateway instance (ADR-013) — with N replicas a team may consume up to N× before blocking.

Guardrail override selects a DIFFERENT Bedrock Guardrail for this team — it cannot remove the provider's configured default (no per-team opt-out, D6/ADR-019).

Allowed regions restrict this team to providers labeled with one of these regions — a provider with NO region label is always blocked for a restricted team (fail-closed, D7/ADR-020).

Submitting this team form replaces any config-declared region policy for that team unless you include the allowed regions here.

add a provider · config block 
// providers: one block per upstream; secrets stay as env/file refs.
"providers": {
  "anthropic-direct": { "type": "anthropic",
    "base_url": "https://api.anthropic.com",
    "api_key_ref": { "env": "ANTHROPIC_API_KEY" } },
  "vllm-internal":    { "type": "openai_compatible",
    "base_url": "http://vllm.internal:8000/v1",
    "api_key_ref": { "file": "/secrets/vllm-key" } },
  "bedrock-us":       { "type": "bedrock", "region": "us-west-2",
    "auth": { "mode": "irsa" } }
},
"models": {
  "claude-sonnet-4-6": { "targets": [
    { "provider": "anthropic-direct", "model": "claude-sonnet-4-6" },
    { "provider": "bedrock-us", "model": "anthropic.claude-sonnet-4-6-v1:0", "api": "invoke_model" }
  ] }
}
