team governance

Rate/TPM/quota/budget limits are enforced live on every request for this team the moment you save — no restart. RPM/TPM/쿼터/예산 한도는 저장 즉시 이 팀의 모든 요청에 실시간으로 적용됩니다 — 재시작이 필요 없습니다.

Counters are enforced per gateway instance (ADR-013) — with N replicas a team may consume up to N× before blocking. 카운터는 인스턴스별로 집계됩니다(ADR-013) — 레플리카가 N개면 차단 전까지 최대 N배까지 소비될 수 있습니다.

Upstream providers and model routing are declared in config (policy-as-code, ADR-003) — this view is read-only. Secrets never appear here: only the env var / file path the gateway reads, or the IAM mode. 업스트림 프로바이더·모델 라우팅은 config로 선언합니다(읽기 전용). 시크릿 값은 표시되지 않고, 참조 이름/IAM 모드만 보입니다.

This gateway has a provider store — you can register and edit providers/routes here (ADR-008). You register only the reference a provider authenticates with (an env var name or a file path) — never the secret value; set the value in your platform's secret store. Changes apply immediately (hot-reload). 프로바이더 스토어가 있어 여기서 등록·편집할 수 있습니다. 시크릿 값이 아니라 참조 이름(env/파일 경로)만 등록하세요 — 값은 본인 시크릿 스토어에 둡니다.

Targets are tried in order (target 1 = primary, the rest = fallback). api is optional (bedrock: invoke_model / converse).

add a provider · config block 
// providers: one block per upstream; secrets stay as env/file refs.
"providers": {
  "anthropic-direct": { "type": "anthropic",
    "base_url": "https://api.anthropic.com",
    "api_key_ref": { "env": "ANTHROPIC_API_KEY" } },
  "vllm-internal":    { "type": "openai_compatible",
    "base_url": "http://vllm.internal:8000/v1",
    "api_key_ref": { "file": "/secrets/vllm-key" } },
  "bedrock-us":       { "type": "bedrock", "region": "us-west-2",
    "auth": { "mode": "irsa" } }
},
"models": {
  "claude-sonnet-4-6": { "targets": [
    { "provider": "anthropic-direct", "model": "claude-sonnet-4-6" },
    { "provider": "bedrock-us", "model": "anthropic.claude-sonnet-4-6-v1:0", "api": "invoke_model" }
  ] }
}

Without a provider store, config is loaded at boot — edit config and reload (kill -HUP) to apply. Set the referenced secret in your platform's secret store — never inline.
